Comparing old fashion typesetting with...

Desktop Publishing

Some examples of desktop publishing are Microsoft Word, Adobe InDesign, and even Google Docs. They are different from old fashioned typesetting because desktop publishing is digital and typesetting is analogue. One way typesetting could be better is that the user can place text wherever they like. But one downside is that it would be hard to align text, which desktop publishing automatically does.

HTML

Html is a little more similar to typesetting, allowing the user to play around with the layout a little more than desktop publishing. The downside of using html is that there is no built in preview whereas in typesetting the user can see right away what it will look like.

CSS

Css is very similar to typesetting as it allows the user to configure fonts, colors, and images. Css is better because it allows users to paste in images of their liking instead of having to make a stamp of it or draw it.

In conclusion, technology regarding writing has come very far and the user experience has gotten easier as well. Though typesetting is obsolete today, it can be appreciated as a classic design of some sorts while digital typesetting can be used for ease and convenience.
